1、eclipse官网  下载安装eclipse的安装工具类   
	网址：https://www.eclipse.org/downloads/
2、下载完毕 点击eclipse的安装工具，进入以后选择eclipse for java ee的版本 ，不要选择java se版本
	【备注：经核实 STS 工具  eclipse的se ee版本都使用此类方法】

3、java ee版本eclipse安装完毕，进入help-install new software
	输入连接：http://dist.springsource.com/release/TOOLS/update/e4.9/     【4.9：为当前eclipse的版本号】 点击回车  ---？选择带有idea的四项  并去掉下面的最后一个复选框，安装完毕重启即可
	
4、检查：进入xml的配置文件，class中输入类名称的前部分然后使用alt+/方式即可查看到提示的类名称，不能使用包路径org.的方式进行提示。
